## 💀 ОПИСАНИЕ ОБЪЕКТА
**VOID TERMINAL** — это зашифрованный сетевой протокол, замаскированный под игру. Вы подключаетесь к терминалу, который анализирует вашу удачу и волю к жизни. Это битва за выживание в эстетике проклятого ЭЛТ-монитора.

### 🧩 ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ
* **CORE:** Чистый JavaScript (Zero dependencies).
* **SYNC:** Real-time синхронизация через **Firebase**.
* **UI:** CRT-эффекты, динамические помехи, кровавая палитра.
* **AI:** Встроенный модуль «TERMINAL» для одиночных сеансов.
